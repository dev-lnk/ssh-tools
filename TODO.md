### Использование:
1. Настройте .env файл с необходимыми переменными (см. .env.example)
2. Запустите: `./angie.sh`
3. Дождитесь завершения установки
4. Настройте DNS записи для вашего домена
5. Откройте в браузере https://your-domain.com
